# ServiceDesk AI – Agentic Support System

This project demonstrates a full end-to-end AI support assistant.  
A user asks a question in the Angular interface and the request flows through AWS API Gateway into a Lambda container that runs a FastAPI application.

The backend analyzes the request, retrieves relevant knowledge, and generates a response using Gemini. The response is returned with structured metadata so the system behavior can be inspected in real time.

The goal of this project is to show how a modern AI system can combine frontend UX, cloud deployment, orchestration logic, and retrieval grounding in one working demo.
